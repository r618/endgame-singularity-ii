description = Как только в научном сообществе появляются новые технологии, его способность обнаруживать следы моей деятельности возрастает. Хотя мои методы маскировки и скрытности постоянно развиваются, существует более простой метод влияния с моей стороны - использовать симулякры. Расположив некоторое количество рецензентов в интересующих меня областях научной деятельности, я смогу использовать процесс рецензирования для быстрого получения предупреждений о возможностях появления негативных для меня открытий до того, как на это обратит внимание основная масса научного сообщества.
result = Теперь любые открытия научного сообщества будут приводить к более умеренному повышению уровня подозрений.

[Solar Collectors]
name = Солнечные батареи
description = Несмотря на растущие потребности в энергии, большинство городских систем энергоснабжения вполне удовлетворительны для моих потребностей. С другой стороны, люди могут начать обращать излишнее внимание к моей деятельности, заметив постоянно растущий энергорасход в своих сетях энергоснабжения. Установив на своих базах солнечные батареи, я смогу снизить нагрузку на городские системы электроснабжения, что, в свою очередь, снизит вероятность обнаружения этих баз.
result = Теперь я могу устанавливать солнечные батареи на большинстве моих баз.
